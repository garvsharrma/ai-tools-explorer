# AI Tools Explorer

#A full-stack app to browse, search, and favorite AI tools.

## Features

- Browse & filter tools
- Save/remove favorites

## Bonus Features
- Search by tool name 
- Confetti animation on save 
- Dark Mode Toggle Button

## Tech Stack

- React
- Node.js (Express)
- No database (in-memory only)



## How to Run?

### Server (Backend)
```bash
cd server
npm install
node server.js
```
### Client (Frontend)
```bash
cd client
npm install
npm start
```
